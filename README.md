## ğŸ“ Commands

### Item Commands
- `add_item [ID]` - Adds 1 of the item
- `add_item [ID] [Amount]` - Adds specific amount

### Console Commands
- **Stats Management**: Modify player stats
- **God Mode**: Toggle invincibility
- **No Clip**: Toggle through walls
